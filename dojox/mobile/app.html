_FormValueWidget

Each _FormValueWidget represents a single input value, and has a (possibly hidden) element, to which it serializes it's input value, so that form submission (either normal submission or via FormBind?) works as expected.

_FormWidget

Represents a single HTML element. All these widgets should have these attributes just like native HTML input elements. You can set them during widget construction or afterwards, via dijit._WidgetBase.attr.

They also share some common methods.
