dojox.mobile.app._FormWidget

Object » dijit._WidgetBase » dojox.mobile.app._FormWidget
dojo.require("dojox.mobile.app._FormWidget");
Defined in dojox.mobile/app/_FormWidget.js

Represents a single HTML element. All these widgets should have these attributes just like native HTML input elements. You can set them during widget construction or afterwards, via dijit._WidgetBase.attr.

They also share some common methods.
